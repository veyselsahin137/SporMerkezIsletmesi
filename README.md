# SporMerkezIsletmesi
Spor Salonu Üyelik ve Çalışan Bilgi Sistemi

Bu proje, Sakarya Üniversitesi Web Programlama dersi kapsamında geliştirilmiş bir Spor Salonu Üyelik ve Çalışan Bilgi Sistemi uygulamasıdır.

Uygulama, ASP.NET Core MVC, Entity Framework Core, LINQ, Bootstrap 5 ve SQL teknolojileri kullanılarak geliştirilmiştir.

Projenin Amacı
Bu projenin amacı, spor salonlarının hizmetlerini, eğitmenlerini, üyelerini ve randevu süreçlerini yönetebilen modern bir web uygulaması geliştirmektir. Ayrıca sistem, üyeler için yapay zekâ tabanlı egzersiz ve diyet önerileri sunan bir modül içermektedir.

Temel Özellikler
Spor salonu ve hizmet tanımlamaları
Eğitmen (antrenör) yönetimi
Üye kayıt sistemi
Randevu oluşturma ve onay mekanizması
REST API üzerinden LINQ ile filtreleme işlemleri
Yapay zekâ entegrasyonu ile kişiselleştirilmiş egzersiz/diyet önerileri
Rol bazlı yetkilendirme (Admin / Üye)
CRUD işlemleri (Create, Read, Update, Delete)
Kullanılan Teknolojiler
ASP.NET Core MVC
C#
Entity Framework Core
SQL Server / PostgreSQL
LINQ
Bootstrap 5
JavaScript / jQuery
OpenAI API (Yapay zekâ öneri sistemi)
